#+TITLE: python typing

+ Tag :: [[id:4F6D630D-B09A-4987-B755-81D9F78656CE][Python]]

Python 从 3.5 版本开始引入了 typing 模块用于增强 *type annotation* 能力，在不影响代码的运行的前提下，可以通过 type annotation 方便 IDE 和开发人员理解代码，
增强可维护性。
